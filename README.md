# Loadstand

Система автоматизации нагрузочного, smoke и стабильностного тестирования.

## Быстрый старт

1. Перейдите в браузере: `http://192.168.2.128:4500`
2. Логин / пароль (по умолчанию):
   - `admin` / `admin123` (или те, что указаны в `backend/storage.py`)
3. Выберите вкладку (smoke / loading / stability)
4. Нажмите **«Запуск»**

Результат выполнения теста появится внизу.

## Возможные проблемы

- Тесты выполняются на сервере, где запущен Loadstand (пока).
- Убедитесь, что все зависимости для тестов установлены (`pip install -r requirements.txt` в папках тестов).
